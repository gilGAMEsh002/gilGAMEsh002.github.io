从头看见尾（第一集）

发表于 2022-10-13 更新于 2022-10-17 分类于 动漫 ， 赛博朋克 阅读次数： 本文字数： 4.1k 阅读时长 ≈ 4 分钟

其实在看第一集，相信大家都已经很快就认识到了赛博朋克世界几个明显的特点–”低科技高生活“、”贫富差距巨大“

从大卫离家上学，这一路就给观众展现了赛博朋克这个世界的样貌，也让观众对大卫的生活有了了解

大卫家很明显并不富裕，洗衣服的钱都会有缴不上的情况，周边的环境也很差，门口就有流浪汉，街上就是抢劫，昨晚喝嗨的人还在吐个不停，光天化日甚至有人玩杯子，孟母看了连夜都要带着儿子跑路。

那么大卫的母亲葛洛丽亚在干什么呢？
观众第一眼看见的葛洛丽亚是躺在沙发上的，大卫见了，虽然抱怨了一下母亲没有给洗衣机交钱，但还是上去给母亲盖衣服，结果葛洛丽亚马上就惊醒了，这里看完之后回来品是让人更心疼母亲的，记得那个医生说“病人劳累过度，精神不太好”，而且这已经不是葛洛丽亚第一次睡沙发了。当然这里也可以是对葛洛丽亚偷走了斯安威斯坦的伏笔，因为大卫动了藏着斯坦威的衣服，所以葛洛丽亚惊醒了。

不管是哪种解释，葛洛丽亚都让人很心疼。

大卫为了省钱用了黑医做的系统导致破坏了学校的系统，葛洛丽亚没怎么睡就又得来学校，面对一笔巨额赔偿。![Cyberpunk - Edgerunners - S01E01 - DUAL 1080p WEB H.264 -NanDesuKa (NF).mkv_000912.917](第一集奠定的基调/Cyberpunk - Edgerunners - S01E01 - DUAL 1080p WEB H.264 -NanDesuKa (NF).mkv_000912.917.png)

这里是一个从校长手部特写开始后退的镜头，一开始就给观众展现一双金光闪闪的手，葛洛丽亚正在道歉，随着镜头后退，我们看到了是一个台灯照在校长手上，手是义手还是什么的，好像镶了金，我将此理解为一种显示权威、地位、财力的方式，而随着后退我们也看到了校长的表情，带着担忧似的，背靠的是高楼大厦
，坐在他面前的葛洛丽亚和大卫背对着观众，葛洛丽亚维持着母亲的尊严，大卫作为犯错的孩子垂着头，虽然他们家今天连洗衣机的钱都没交，但葛洛丽亚还是沉着地表示一定会做出赔偿。

当校长表示大卫不适合这所学校，建议转校时，葛洛丽亚坚决地说![Cyberpunk - Edgerunners - S01E01 - DUAL 1080p WEB H.264 -NanDesuKa (NF).mkv_000950.762](第一集奠定的基调/Cyberpunk - Edgerunners - S01E01 - DUAL 1080p WEB H.264 -NanDesuKa (NF).mkv_000950.762-1665933977046.png)

这里的镜头给了大卫不耐烦的侧脸，而葛洛丽亚只能看到下巴，让人感觉对儿子有些冷漠。

随后原封不动的镜头直接转场，葛洛丽亚开车带大卫离开，场景变了但人物情绪还没有![Cyberpunk - Edgerunners - S01E01 - DUAL 1080p WEB H.264 -NanDesuKa (NF).mkv_000956.801](第一集奠定的基调/Cyberpunk - Edgerunners - S01E01 - DUAL 1080p WEB H.264 -NanDesuKa (NF).mkv_000956.801.png)

这里的母亲既展现出了柔弱的部分——她所做的一切都是为了儿子，又展现坚强的部分——即使现实如此糟糕依然相信儿子会到达‎爬到荒坂塔的顶端。伸出一根手指说着”爬到荒坂塔的顶端“的葛洛丽亚，阳光洒在了她的脸上，而转过头来我们看清了她脸上的憔悴。

这里葛洛丽亚说话时车窗外的情形已经暗示了后面发生的战斗，黑色高级轿车被一伙混混炸翻，转而又撞上了他们的车。下一个镜头就是一个颠倒的镜头，大卫被卡在翻倒后破损的车里出不来，葛洛丽亚被甩到了路边不省人事。这时，在前面背景出现过几次的创伤小组赶来了，，，

下一个镜头直接就是医院的安全出口标识，表示大卫带着他母亲出来了，绿色的安全出口标识让人觉得安心，但很快这个灯就闪了一下，蹦出了火花，暗示安全是不稳定的。手术成功了，但是不允许探病，得加钱。

大卫只能先回家去准备钱，bgm变成了开头大卫出门上学时的有些滑稽的bgm：Who’s Ready for Tomorrow，但大卫回去时房屋租金缴纳逾期了（葛洛丽亚甚至没来得及缴），只能从排气口进去。大卫在家里准备钱的时候，给了两件叠好的衣服的镜头，应该是葛洛丽亚给洗衣机交了钱洗好了大卫的衣服，可以看出葛洛丽亚作为母亲对大卫的事很上心。从这里开始出现了很多的对前面的对比，比如大卫再次走在街道上，再次回首找寻那抹绚丽的色彩，但看到的是同学胜男嘲讽的脸。

在被胜男狠狠地嘲讽，并被他高级的义体痛揍一顿后，大卫接到了医生的电话

随后又是一段对前面大卫上学路的照应、对比，不同的是大卫这次低着头，双手端着母亲的骨灰，催促付钱的电话一个接一个。社会对这个刚刚丧母的少年露出了吸血的獠牙。

就此大卫被逼上了成为赛博边缘人的道路。
